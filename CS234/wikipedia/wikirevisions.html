TASK for you: Unpack this expression str(createDateTime(rev).date()) to understand what is happening.

The function createDateTime(rev) takes in a revision dictionary (rev), and gets the timestamp. It converts it from a timestruct object to a datetime object and returns that datetime object representing the timestamp. The expression str(createDateTime(rev).date()), gets the datetime version of the timestamp in the revision rev and extracts the date with the function .date() and turns it into a string using str()
